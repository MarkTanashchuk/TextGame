# Text Adventure "Кулю в лоб чи в Гааґу гоп"

## Prerequisites

- Clone the repo

```cli
git clone https://github.com/MarkTanashchuk/TextGame && cd ./TextGame
```

- Install dependencies

```cli
npm run template-install
```

## Usage

Start server

```cli
npm run start
```

Open [localhost:3000](http://localhost:3000)
